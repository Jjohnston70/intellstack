## Appendix: Metric Definitions

| Metric | Definition | How to Calculate |
|--------|------------|------------------|
| Lead → Sale Conversion | % of qualified leads that purchase assessment | Assessments Sold ÷ Qualified Leads |
| Assessment → Implementation | % of completed assessments that buy implementation | Implementation Deals ÷ Completed Handoffs |
| Avg Assessment Value | Average revenue per assessment sold | Total Assessment Revenue ÷ Count Sold |
| Avg Implementation Value | Average implementation contract value | Total Implementation Revenue ÷ Count Sold |
| Revenue per Assessment | Total revenue generated per assessment (including implementation) | (Assessment Rev + Implementation Rev) ÷ Assessments |
| Cycle Time | Days from lead creation to deal close | Close Date - Lead Date |
| First-Pass Approval | % of assessments approved at peer review without revision | Approved First Time ÷ Total Reviewed |
| Nurture Reactivation | % of "not now" clients who later convert | Reactivated ÷ Total Nurture |
