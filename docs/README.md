## Legal Templates (Local-Only)

The `legal/` subfolder contains draft legal templates for IP protection. These are excluded from version control to protect trade secrets.

**Available templates:**
- `NDA-TEMPLATE-STANDARD.md` - Full NDA for contractors/partners
- `NDA-TEMPLATE-MUTUAL.md` - Bidirectional NDA for partnerships
- `NDA-TEMPLATE-PROSPECT.md` - Simple acknowledgment for sales calls
- `IP-CLAUSES-CLIENT-AGREEMENT.md` - IP clauses for client contracts
- `IP-CLAUSES-CONTRACTOR.md` - Work-for-hire and IP assignment clauses

**Important:** All templates require attorney review before use.
