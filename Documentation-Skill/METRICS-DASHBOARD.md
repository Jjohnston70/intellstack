## Part 3: Efficiency Metrics

### Time to Delivery

Track how long documents take from request to delivery.

| Document Type | Target Time | Actual Average | Variance |
|---------------|-------------|----------------|----------|
| Service Document | 45 min | __ min | __% |
| Proposal | 60 min | __ min | __% |
| SOP | 60 min | __ min | __% |
| Cheat Sheet | 35 min | __ min | __% |
| Agreement | 45 min | __ min | __% |

**Calculation:**
```
Variance = (Actual - Target) / Target × 100
```

**Interpretation:**
- Positive variance: Taking longer than expected
- Negative variance: Faster than expected
- >20% variance: Investigate cause

## Quick Reference

### Key Metrics to Watch

1. **Revision rate** — Are documents right first time?
2. **Time to delivery** — Are we efficient?
3. **Proposal acceptance** — Are proposals effective?
4. **Checklist completion** — Is quality process followed?

### Red Flags

| Signal | Possible Cause | Action |
|--------|---------------|--------|
| Revision rate >15% | Template gaps, unclear requirements | Review revision reasons |
| Time variance >30% | Scope creep, information gaps | Improve intake process |
| Checklist completion <80% | Process not followed | Reinforce training |
| Brand compliance <90% | Templates outdated | Update templates |
