# Direction Protocol Visual Reference

## The Complete Flow

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                         THE DIRECTION PROTOCOL                                  │
│                    From Lead to Launch in 5 Stages                              │
└─────────────────────────────────────────────────────────────────────────────────┘

┌─────────────┐
│   LEAD      │  Sources: Referral, Inbound, Networking, Cold Outreach
│  ARRIVES    │
└──────┬──────┘
       │
       ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│  STAGE 1: IDENTIFY (15-20 minutes)                                              │
├─────────────────────────────────────────────────────────────────────────────────┤
│  Purpose: Is there a fit?                                                       │
│                                                                                 │
│  Process:                                                                       │
│  1. Opening based on lead source                                               │
│  2. Ask 3 core questions                                                       │
│     - Tell me about your operation                                             │
│     - What keeps you up at night?                                              │
│     - Magic wand - what would you fix?                                         │
│  3. Listen for pain signals vs disqualifiers                                   │
│  4. Run qualification check (4 criteria)                                       │
│                                                                                 │
│  Output:                                                                        │
│  ├─ GREEN (7-8/8): Schedule ASSESS → ────────────┐                            │
│  ├─ YELLOW (5-6/8): Schedule ASSESS with notes ──┤                            │
│  ├─ ORANGE (3-4/8): Pipeline Punks option ───────┤                            │
│  └─ RED (0-2/8 or disqualifiers): Walk away ─────┼─────→ [END]                │
│                                                   │                            │
└───────────────────────────────────────────────────┼────────────────────────────┘
                                                    │
                                                    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│  STAGE 2: ASSESS (30 minutes)                                                   │
├─────────────────────────────────────────────────────────────────────────────────┤
│  Purpose: Is the problem real and solvable?                                     │
│                                                                                 │
│  Process:                                                                       │
│  1. Operational Reality (10 min)                                               │
│     - Walk through typical workflow                                            │
│     - Where things go wrong                                                    │
│     - How they find out about problems                                         │
│     - What decisions they're guessing on                                       │
│                                                                                 │
│  2. Current State (8 min)                                                      │
│     ┌────────────────────────────┐                                            │
│     │  >>> USE SRO ASSESSMENT <<<│                                            │
│     └────────────────────────────┘                                            │
│     - Where data lives now                                                     │
│     - How they find information                                                │
│     - Who knows how things work                                                │
│                                                                                 │
│  3. Stakes and Readiness (5 min)                                               │
│     - Cost of status quo                                                       │
│     - One year projection                                                      │
│     - Vision for "fixed"                                                       │
│                                                                                 │
│  4. Fit Check (3 min)                                                          │
│     - Explain MAP requirement (90 min)                                         │
│     - Gauge commitment                                                         │
│                                                                                 │
│  5. Budget Reality (2 min)                                                     │
│     - Budget range discussion                                                  │
│                                                                                 │
│  Output:                                                                        │
│  ├─ GREEN: Schedule MAP (90-120 min) ────────────┐                            │
│  ├─ YELLOW: Pipeline Punks alternative ──────────┤                            │
│  └─ RED: Polite exit ─────────────────────────────┼─────→ [END]                │
│                                                   │                            │
└───────────────────────────────────────────────────┼────────────────────────────┘
                                                    │
                                                    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│  STAGE 3: MAP (90-120 minutes)                                                  │
├─────────────────────────────────────────────────────────────────────────────────┤
│  Purpose: What's broken? What's needed? How much?                               │
│                                                                                 │
│  Process:                                                                       │
│  1. Introduction (5 min)                                                       │
│  2. Current State Mapping (30 min)                                             │
│     ┌────────────────────────────┐   ┌──────────────────────────┐            │
│     │  >>> USE SRO ASSESSMENT <<<│   │  >>> USE WORLD MAPPER <<<│            │

│     - Decision points identified                                               │
│     - Bottlenecks documented                                                   │
│     - Systems inventory created                                                │
│                                                                                 │
│  3. Pain Point Deep Dive (20 min)                                              │
│     - Root cause analysis                                                      │
│     - Impact quantification                                                    │
│                                                                                 │
│  4. Desired State (15 min)                                                     │
│     - Vision definition                                                        │
│     - Capabilities needed                                                      │
│     - Metrics to track                                                         │
│                                                                                 │
│  5. Solution Architecture (15 min)                                             │
│     - Command modules selection                                                │
│     - Integration requirements                                                 │
│     - Implementation approach                                                  │
│                                                                                 │

│  7. Next Steps (5 min)                                                         │
│                                                                                 │
│  ┌──────────────────────────────────────────────────────────┐                 │
│  │  PRICING CALCULATION (automatic)                         │                 │
│  ├──────────────────────────────────────────────────────────┤                 │
│  │  Score 5 factors (1-3 each):                             │                 │
│  │  1. Crew/Employee count                                  │                 │
│  │  2. System chaos level                                   │                 │
│  │  3. Data source count                                    │                 │
│  │  4. Owner availability                                   │                 │
│  │  5. Technical comfort                                    │                 │
│  │                                                           │                 │
│  │  Total 5-7 → $2,500 base                                 │                 │
│  │  Total 8-11 → $3,500 base                                │                 │
│  │  Total 12-15 → $5,000 base                               │                 │
│  │                                                           │                 │
│  │  + Adjustments ($500-1000 each):                         │                 │
│  │    • Multiple locations                                  │                 │
│  │    • Field AND office ops                                │                 │
│  │    • Custom reporting                                    │                 │
│  │    • Tight timeline                                      │                 │
│  └──────────────────────────────────────────────────────────┘                 │
│                                                                                 │
│  Output:                                                                        │
│  ├─ Complete operational map (current + desired)                               │
│  ├─ Pricing calculation with reasoning                                         │
│  ├─ Command modules list                                                       │
│  ├─ Implementation timeline                                                    │
│  └─ Schedule CHART presentation ─────────────────┐                            │
│                                                   │                            │
└───────────────────────────────────────────────────┼────────────────────────────┘
                                                    │
                                                    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│  STAGE 4: CHART (30-45 minutes)                                                 │
├─────────────────────────────────────────────────────────────────────────────────┤
│  Purpose: Present findings and get commitment                                   │
│                                                                                 │
│  Presentation Structure:                                                        │
│  1. The Current Reality (5 min)                                                │
│     "Here's what we heard..."                                                  │
│     [Show operational map from World Mapper]                                   │
│                                                                                 │
│  2. The Problems (15 min)                                                      │
│     "Here's what's broken and why..."                                          │
│     [Root causes + impact]                                                     │
│                                                                                 │
│  3. The Solution (10 min)                                                      │
│     "Here's what we'll build..."                                               │
│     [Command modules + approach + timeline]                                    │
│                                                                                 │
│  4. The Investment (10 min)                                                    │
│     "Here's what it costs and what you get..."                                 │
│     [Price + payment terms + deliverables]                                     │
│                                                                                 │
│  5. The Outcome (5 min)                                                        │
│     "Here's what changes..."                                                   │
│     [Specific wins addressing their pain]                                      │
│                                                                                 │
│  Handle Objections:                                                             │
│  ├─ "That's expensive" → Compare to cost of status quo                         │
│  ├─ "Need to think" → What questions would help you decide?                    │
│  ├─ "Found cheaper" → Cheaper doesn't mean it solves the problem              │
│  └─ "Just tell me price" → [Already past this - you earned the right]         │
│                                                                                 │
│  Output:                                                                        │
│  ├─ YES: Move to LAUNCH ─────────────────────────┐                            │
│  ├─ NEGOTIATE: Adjust scope/price/timeline ──────┤                            │
│  └─ NO: Understand why, offer alternatives ──────┼─────→ [END]                │
│                                                   │                            │
└───────────────────────────────────────────────────┼────────────────────────────┘
                                                    │
                                                    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│  STAGE 5: LAUNCH                                                                │
├─────────────────────────────────────────────────────────────────────────────────┤
│  Purpose: Execute contract and begin delivery                                   │
│                                                                                 │
│  Process:                                                                       │
│  1. Sign service agreement                                                     │
│  2. Receive deposit (50% or first payment)                                     │
│  3. Schedule kickoff meeting                                                   │
│  4. Assign delivery team                                                       │
│  5. Set up project tracking                                                    │
│  6. Begin Command Protocol (delivery)                                          │
```

## Red Flag vs Green Flag Matrix

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                          Qualification Matrix                                   │
└─────────────────────────────────────────────────────────────────────────────────┘

RED FLAGS (Walk Away)              │  GREEN FLAGS (Good Fit)
──────────────────────────────────┼──────────────────────────────────────
                                   │
Won't commit to Map stage          │  Shows up on time, prepared
Wants price before assessment      │  Asks good questions
Wants guaranteed outcomes          │  Takes notes
Argues before understanding scope  │  Shares detailed info freely
Expects you to manage team         │  Acknowledges their role in problems
"Tried everything" mentality       │  Open to changing approaches
Blames everyone else               │  Sees value beyond price
Disrespectful of time              │  Decision maker engaged
Bad vibe / dread working           │  Team is bought in
Keeps rescheduling                 │  Has tried to solve before
Won't answer direct questions      │  Respects the process
Unwilling to change anything       │  Follows through on commitments
Looking for silver bullet          │  Strong referral source
Not decision maker (won't loop in) │  Good industry expertise
Budget severely misaligned         │  Growth trajectory

┌──────────────────────────────────────────────────────────────┐
│  SCORING GUIDE:                                              │
├──────────────────────────────────────────────────────────────┤
│  2+ Red Flags  → Seriously consider walking away            │
│  4+ Red Flags  → Walk away immediately                      │
│  6+ Green Flags → High probability close                    │
│  8+ Green Flags → Ideal client, prioritize                  │
└──────────────────────────────────────────────────────────────┘
```

## Pricing Quick Reference

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                        Command Center Build Pricing                             │
└─────────────────────────────────────────────────────────────────────────────────┘

FACTOR 1: Crew/Employee Count        FACTOR 2: System Chaos
├─ 5-8 employees ──► 1 point         ├─ Organized spreadsheets ──► 1 point
├─ 9-14 employees ─► 2 points        ├─ Paper + texts + scattered ► 2 points
└─ 15-20 employees ► 3 points        └─ Complete chaos ─────────► 3 points

FACTOR 3: Data Source Count          FACTOR 4: Owner Availability
├─ 2-3 sources ────► 1 point         ├─ Highly available ────────► 1 point
├─ 4-5 sources ────► 2 points        ├─ Busy but makes time ─────► 2 points
└─ 6+ sources ─────► 3 points        └─ Hard to pin down ────────► 3 points

FACTOR 5: Technical Comfort
├─ Quick learner ──────────► 1 point
├─ Needs support ──────────► 2 points
└─ Resistant ──────────────► 3 points

┌────────────────────────────────────────────────────────────────┐
│  BASE PRICING:                                                 │
├────────────────────────────────────────────────────────────────┤
│  Total Score 5-7   → $2,500 base                              │
│  Total Score 8-11  → $3,500 base                              │
│  Total Score 12-15 → $5,000 base                              │
└────────────────────────────────────────────────────────────────┘

ADJUSTMENTS (ADD $500-1000 each):
├─ Multiple locations
├─ Both field AND office operations
├─ Custom reporting needed
└─ Tight timeline

ADJUSTMENTS (SUBTRACT $500 each):
├─ Some systems already working
├─ Very small operation
└─ Strong referral

┌────────────────────────────────────────────────────────────────┐
│  PAYMENT TERMS:                                                │
├────────────────────────────────────────────────────────────────┤
│  Under $4,000  → 50% start / 50% completion                   │
│  Over $4,000   → 40% start / 30% midpoint / 30% completion    │
└────────────────────────────────────────────────────────────────┘
```

## Quick Command Reference

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                    Say This → Claude Does This                                  │
└─────────────────────────────────────────────────────────────────────────────────┘

"New lead from [source]"              → Start IDENTIFY stage
"Score this prospect"                 → Run qualification scorecard
"ASSESS call scheduled"               → Prepare ASSESS materials
"Start MAP session"                   → Structure 90-min deep dive
"Calculate pricing"                   → Run complexity scoring
"Generate proposal"                   → Create CHART presentation
"Should we take this client?"         → Analyze fit signals
"What are the red flags?"             → List concerns
"Create service agreement"            → Generate contract
"Client signed"                       → Initiate LAUNCH
"Track this in pipeline"              → Add to pipeline tracking
"What's next?"                        → Identify next action
```
