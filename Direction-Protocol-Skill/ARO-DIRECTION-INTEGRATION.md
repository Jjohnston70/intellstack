## Overview

ARO Assessment is a **productized version** of Direction Protocol's ASSESS + MAP stages, focused specifically on AI/automation readiness. This document explains when to use each, how they integrate, and how to manage clients through both.

**Key Insight:** ARO Assessment IS Direction Protocol ASSESS+MAP — just packaged as a standalone product with specific deliverables.

### What ARO Assessment Produces

| Context Architecture Report | Score + gaps analysis | MAP findings |

| Context Graph v1.0 | Consolidated client context | Would be created anyway |

### ARO → Implementation Transition

When an ARO Assessment client says YES to implementation:

1. **Skip re-doing MAP** — ARO findings ARE the MAP findings
2. **Apply $500 credit** — Reduces implementation investment
3. **Go directly to CHART** — Present implementation proposal
4. **Context Graph already exists** — Upgrade to v2.0, don't recreate

**Script for ARO → CHART:**
> "Since we've already done the deep diagnostic with your ARO Assessment, I have everything I need to recommend a solution. Based on your score of [X.X] and what we found, I recommend [Service]. The investment is $[X,XXX] — and remember, your $500 assessment credit applies, bringing it to $[X,XXX - 500]."

## Part 3: Direction Protocol with ARO Elements

When a prospect enters via full Direction Protocol but has AI/automation aspirations.

### When to Recommend ARO During ASSESS or MAP

**During ASSESS, if you hear:**
- "We want to use AI but don't know where to start"
- "Our automations keep breaking"
- "ChatGPT/Copilot doesn't understand our business"
- "We're not sure if we're ready for AI"

**Script:**
> "It sounds like you're interested in AI and automation, but not sure where to start. We have a diagnostic called the ARO Assessment that's designed exactly for this. It evaluates your operational readiness for AI tools and shows you exactly where the friction is. Investment is $1,500-$2,500 and takes about a week. Interested?"

**If YES:** Pivot to ARO Assessment path
**If NO:** Continue with standard Direction Protocol

### During MAP, when to use ARO Framework

Even if the prospect didn't buy a standalone ARO Assessment, you can use the ARO scoring framework during MAP to evaluate their AI readiness:

**ARO Scoring Areas:**
1. File Organization (1-5)
2. Process Documentation (1-5)
3. Data Structure (1-5)
4. Decision Logic (1-5)
5. Communication Records (1-5)
6. Access & Permissions (1-5)

**Score Interpretation:**
- 4.0-5.0: Agent-ready — AI tools will work well
- 3.0-3.9: Functional — Some friction, fixable
- 2.0-2.9: Poor — Significant barriers to AI success
- 1.0-1.9: Chaotic — Need foundational work first

**Include in MAP findings:**
> "Based on my assessment, your ARO readiness score is approximately [X.X]. This means [interpretation]. Before AI tools will work consistently, we need to address [top friction points]."

### When ARO Clients Return for Implementation

If an ARO-only client returns later for implementation:

1. **Check Context Graph v1.0** — Is it still current?
2. **Quick validation call (30 min)** — What's changed since assessment?
3. **Update findings if needed** — Major changes may require re-assessment
4. **Jump to CHART** — No need for full MAP
5. **Apply $500 credit** — Still valid (90-day standard, extend if needed)

**Script for returning ARO client:**
> "Great to hear from you! I have your ARO Assessment findings from [date]. A few quick questions: Has anything major changed in your operation since then? [If no:] Perfect — I can put together an implementation proposal based on what we found. The $500 credit still applies."

## Part 7: Common Scenarios

### Scenario A: ARO Assessment → YES at Handoff

**Situation:** Client completed ARO Assessment and says YES to implementation

**Actions:**
1. Send implementation proposal (same day)
2. Apply $500 credit
3. Agreement signed
4. Move files to `30_ACTIVE_CLIENTS/`
5. Upgrade Context Graph v1.0 → v2.0
6. Schedule kickoff
7. → Command Protocol begins

**Skip:** Re-doing ASSESS, re-doing MAP, recreating Context Graph

### Scenario B: ARO Assessment → Needs More Discovery

**Situation:** ARO findings reveal complexity beyond what was assessed

**Script:**
> "Based on what we found, your situation is more complex than we initially scoped. I'd like to do an extended MAP session to dig deeper into [specific areas]. This would be an additional [X hours/$X]. Want to proceed?"

**Actions:**
1. Propose extended discovery (hourly or flat fee)
2. Update ARO findings with new information
3. Revise implementation recommendation if needed
4. Present updated proposal

### Scenario C: Direction Protocol Client Asks About AI

**Situation:** During ASSESS or MAP, client mentions AI aspirations

**Script:**
> "You mentioned wanting to use AI tools. That's great — but AI only works well when your operations are structured to support it. I can evaluate your AI readiness as part of this engagement. Want me to include that in my findings?"

**Actions:**
1. Add ARO scoring framework to MAP session
2. Include AI readiness score in findings
3. Address AI readiness in recommendations
4. If score is low, explain what needs to change first

### Scenario D: Client Wants Both Assessment AND Implementation Quoted

**Situation:** Client wants to know total investment upfront

**Script:**
> "Sure. The ARO Assessment is $[X] and takes about a week. Implementation typically ranges from $2,500 to $8,000 depending on what we find. I can't give you an exact number until we do the assessment, but most clients in your situation end up around $[estimate]. And the $500 credit applies, so think of the assessment as a down payment."

**Ballpark guidance:**
- Simple operation + low ARO score: ~$5,000-$6,500 total
- Medium complexity: ~$7,000-$10,000 total
- Complex operation: ~$10,000-$15,000 total
