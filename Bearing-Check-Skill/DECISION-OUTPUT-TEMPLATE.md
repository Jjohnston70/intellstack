# Bearing Check Decision Output Template

**Version:** 1.0
**Last Updated:** January 29, 2026
**Purpose:** Standardized template for documenting Bearing Check decision analyses

---

## When to Use This Template

Use this template to document the output of a Bearing Check session. This creates a decision record that:
- Captures your analysis for future reference
- Allows you to revisit decisions and assess accuracy
- Provides accountability for decision-making
- Creates learning material for pattern recognition

---
